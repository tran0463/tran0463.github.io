Five HTML tags

Anchor Tag <a>
Anchor tag is used for linking to a site, whether internal or external
Header Tag <header>
Header tag is used to create a section of the file dedicated for the header
Div Tag <div>
Div tag is used to create a section of a file dedicated for content, ususally used in conjunction with IDs/names or classes
Footer Tag <footer>
Opposite of Header Tag, used to create a section of a file for the footer
Meta Tag <meta>
Meta tags are sets of instructions for browsers and search engines on how to handle files. Is a self closing tag so you don't need to end it with </meta>
